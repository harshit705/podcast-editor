🎙️ AI Podcast Editor

An AI-powered Podcast Editor that helps you automatically transcribe, edit, and format podcasts with ease. Built with modern web technologies for speed, scalability, and flexibility.

🚀 Features

🎧 Upload audio/video files for editing

✨ Automatic transcription using AI

🗣️ Speaker diarization for identifying multiple speakers

✂️ Smart silence detection & removal

🎨 Clean and modern UI with Tailwind CSS + shadcn-ui

⚡ Fast development setup with Vite & TypeScript

🌐 Easy deployment on Vercel, Netlify, or your own server

📦 Installation
Prerequisites

Node.js (>= 18)

npm or yarn

Steps
# Clone the repository
git clone https://github.com/your-username/ai-podcast-editor.git

# Navigate into the project
cd ai-podcast-editor

# Install dependencies
npm install

# Start the development server
npm run dev
